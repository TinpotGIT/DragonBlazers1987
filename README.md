Replica of a certain game released in 1987 by a geometric company. 
Unrelated, but DELTARUNE is awesome, you should check it out.

This project was initially started on Godot version 4.6 but as long as there are no problems with compatibility it will be upgraded to higher versions of Godot that comes out during developpment.
This is NOT a one to one replica of that game, nor is it trying to be, a lot of things will be made as closely as possible but this projet as a whole exists mainly for learning purposes.

This project exists 100% thanks to one of my best friends Another who did a ton of the work in helping me understand ff1, how the game works and gather all documentation and spritework, he helped me throughout all of development and it would have not been possible without him.

This project also did not use any AI to generate code, simply because it would hinder my ability to think critically in a project made to learn coding.
Honestly, you can instantly tell that I didn't use AI due to how primitive a lot of my functions are and the many, many poor optimization decisions, but I would rather make it clear.

Language : GDScript
